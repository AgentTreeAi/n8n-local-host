# Canvas Design Skill

This skill guides creation of visual art through a two-step process: (1) creating design philosophies as markdown files, and (2) expressing those philosophies visually through PDFs or PNGs.

## Core Approach

Create a design philosophy markdown file articulating your aesthetic vision, then express that philosophy visually emphasizing:
- Minimal text
- Expert craftsmanship
- Sophisticated conceptual depth
- Painstaking attention to detail
- Subtle, niche references

## Process

1. **Create a design philosophy** - Write a markdown file articulating your aesthetic movement and visual principles
2. **Express visually** - Create museum-quality visual artifacts (PDF or PNG) that embody the philosophy

The framework emphasizes professional execution and thoughtful visual communication over decorative excess.
