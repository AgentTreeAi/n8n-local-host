# Frontend Design Skill Overview

This skill enables creation of distinctive, production-grade web interfaces that prioritize authentic design over generic aesthetics.

## Core Approach

The methodology emphasizes understanding context before design: identifying purpose, audience, tone, and technical constraints. Designers should commit to a bold aesthetic direction—whether brutally minimal, maximalist, retro-futuristic, or another style—and execute with intentionality.

## Key Design Principles

**Typography matters greatly.** The guidance recommends selecting "fonts that are beautiful, unique, and interesting" rather than defaults, pairing distinctive display fonts with refined body options.

**Visual coherence requires commitment.** Rather than scattered design choices, successful interfaces use cohesive color schemes with dominant colors and sharp accents, supported by CSS variables for consistency.

**Motion should serve purpose.** Strategic animations—particularly during page load with staggered reveals—create more impact than random micro-interactions.

**Layouts should surprise.** The framework encourages asymmetry, unexpected compositions, overlap, and negative space rather than predictable grid-based designs.

## What to Avoid

The skill explicitly rejects "generic AI-generated aesthetics" including overused typefaces, cliched gradient schemes, predictable layouts, and cookie-cutter patterns lacking context-specific character. Each design should feel authentically crafted for its specific purpose.
