# Internal Communications Skill Overview

This skill helps you create company-specific internal communications across multiple formats. It's designed to streamline writing for common workplace documents.

## Key Communication Types Supported

The skill covers seven main communication categories: 3P updates (Progress, Plans, Problems), company newsletters, FAQ responses, status reports, leadership updates, project updates, and incident reports.

## How It Works

The process is straightforward—first identify what type of communication you need, then reference the matching guideline file from the `examples/` directory. There are four template files available: one for 3P team updates, one for company-wide newsletters, one for FAQ answers, and a general file for other communication needs.

## When to Apply This Skill

This resource becomes useful whenever you're drafting internal-facing documents like weekly team updates, company announcements, responses to common questions, or project status briefings.

The skill emphasizes following company-specific formatting and tone conventions already established in those guideline templates, ensuring consistency across all internal communications.
