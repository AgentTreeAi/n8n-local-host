# Anthropic Brand Styling Documentation

This resource provides Anthropic's official brand identity specifications for visual artifacts. The guidelines include:

**Color Palette:**
The brand defines primary colors as dark (#141413), light (#faf9f5), and mid-gray (#b0aea5), alongside accent colors in orange (#d97757), blue (#6a9bcc), and green (#788c5d).

**Typography Standards:**
"Headings: Poppins (with Arial fallback)" and body text uses Lora with Georgia as fallback. The system automatically applies these fonts with sizes of 24pt and larger for headings.

**Implementation Features:**
The skill intelligently applies brand elements—custom fonts to text elements while cycling accent colors through non-text shapes. It works with pre-installed system fonts and requires no additional font installation.

**Key Capability:**
The system "uses RGB color values for precise brand matching" while maintaining visual consistency across different platforms and systems.

This resource is designed for designers and developers needing to apply consistent Anthropic visual identity to various project types.
